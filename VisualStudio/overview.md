# LetsCollab VS Theme

LetsCollab VS Theme brings the LetsCollab dark palette to Visual Studio with a focus on readability, consistent syntax highlighting, and cohesive UI accents.

## Highlights

- Consistent dark UI across editor surfaces and tool windows
- Semantic color mappings aligned with Rider and VS Code theme variants
- Regularly synchronized palette and classification updates
